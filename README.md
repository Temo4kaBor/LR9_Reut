# LR9_Reut

Учебный веб-фреймворк на базе Node.js, использующий стандартные модули `http`, `events` (EventEmitter) и `fs`.

## Основные возможности
- **Сборка данных через Streams**: Обработка входящего JSON-потока.
- **Middleware**: Поддержка цепочки промежуточных обработчиков (например, логирование).
- **EventEmitter**: Маршрутизация на основе событийной модели.
- **Persistent Storage**: Сохранение данных в JSON-файлы.

## Сущности данных
Проект работает с двумя типами данных, включающими типы: string, number, boolean, Date, Array.
1. **Подразделения (Units)**: Название, численность, боеготовность, дата учений, снаряжение.
2. **Офицеры (Officers)**: ФИО, уровень звания, дежурство, дата повышения, медали.

## API Маршруты

### Подразделения (Units)
- `GET /units` — Список всех подразделений.
- `GET /units/detail?id=...` — Детальная информация по конкретному ID.
- `POST /units` — Добавление нового подразделения.
- `PATCH /units/update?id=...` — Частичное обновление данных подразделения.

### Офицеры (Officers)
- `GET /officers` — Список всех офицеров.
- `POST /officers` — Добавление нового офицера в базу.
- `DELETE /officers/delete?id=...` — Удаление офицера по ID.

## Как запустить
1. Убедитесь, что в корне есть папка `db` с файлами `units.json` и `officers.json`.
2. Запустите сервер командой: `node index.js`.
3. Тестируйте запросы через Thunder Client или Postman на порту 3000.
